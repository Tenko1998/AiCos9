# Candy Brand Guidelines

Candy is an open-source project, but the Candy name, logo, domain names, and official visual identity are separate from the source code license.

The open-source license applies to the code and other materials explicitly released under that license. It does not automatically grant permission to use the Candy name, logo, or official brand identity in a way that may confuse users about what is official.

## Allowed Uses

You may use the Candy name for factual references, such as:

- "based on Candy"
- "forked from Candy"
- "compatible with Candy"
- "an unofficial Candy fork"

You may also link to the official Candy repository, discuss the project, review it, contribute to it, or build independent work based on the source code under the project license.

## Forks and Modified Versions

Forks and modified versions should use their own names.

A fork may explain that it is based on Candy, but it should not present itself as the official Candy project unless it is maintained or approved by the Candy project owner.

Good examples:

- "MyProject, based on Candy"
- "An unofficial Candy fork"
- "Compatible with Candy"

Avoid examples:

- "Candy Pro" without permission
- "Official Candy" without permission
- using the Candy name, logo, or visual identity in a way that makes a project look official when it is not

## Uses That Need Permission

Please ask for permission before using the Candy name, logo, or official brand identity for:

- a public app, website, package, or hosted service
- a commercial product or paid service
- a social media account or community that may look official
- merchandise, paid assets, paid plugins, or marketplace listings
- a modified distribution that uses Candy as its main name

## No Confusion Principle

The main rule is simple: users should be able to tell what is official and what is independent.

The Candy brand should not be used in a way that confuses users about who maintains, approves, operates, or is responsible for a project, service, account, or distribution.
